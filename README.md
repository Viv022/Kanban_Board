## 🛠️ Installation & Setup

To get this Kanban board running on your local machine, follow these steps:

**Prerequisites:**
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

**1. Clone the repository**
```bash
git clone [https://github.com/Viv022/Kanban_Board.git](https://github.com/Viv022/Kanban_Board.git)
```

**2. Navigate to the project directory**
```bash
cd Kanban_Board
```

**3. Install dependencies using your preferred package manager**
```bash
npm install
# or 'bun install'
```

**4. Start the development server**
```bash
npm run dev
# or 'bun run dev'
